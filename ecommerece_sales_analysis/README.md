# 🛍️ E-Commerce Sales Analysis

## What This Project Does
Sales data analysis covering product performance,
customer spending patterns and category revenue.

## Skills Used
- **Pandas**: groupby, pivot_table, nlargest, value_counts
- **NumPy**: mean, max, min, std calculations
- **Matplotlib**: bar charts

## Key Insights
- Identified top selling products by revenue and quantity
- Analyzed customer spending tiers (Low, Medium, High)
- Found highest value transactions using boolean filtering

## Files
- basic_analaysis.py — basic analysis
- advance_analysis.py — advanced analysis
