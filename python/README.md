# 🐍 Python Core & DSA Practice

## What This Project Does
Python fundamentals and algorithm problem solving
covering core programming concepts and data structures.

## Topics Covered
- List operations, string manipulation, slicing
- Recursion and nested data structures
- Dictionary based frequency counting
- Stack based algorithms (balanced parentheses)
- Anagram grouping using sorted keys
- Missing number detection using math formula
- Fibonacci series, prime checking
- Sorting algorithms without built-in sort

## Skills
- Python: loops, functions, recursion, list comprehension
- Problem solving: hashmaps, stacks, two-pointer logic

## Files
- practice.py — basic Python problems (Q1-Q17)
- practice_1_7.py — string and list problems
- practice_8_17.py — intermediate problems
