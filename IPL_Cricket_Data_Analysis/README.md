# 🏏 IPL Cricket Data Analysis

## What This Project Does
Analysis of IPL matches from 2008-2024 using ball-by-ball
and match-level data to find performance trends.

## Dataset
- Source: Kaggle — IPL Complete Dataset
- matches.csv: 1095 matches
- deliveries.csv: 260,920 ball by ball records

## Skills Used
- Pandas: merge, groupby, crosstab, pivot_table,
  drop_duplicates, isin, fillna
- NumPy: mean, std, percentile, np.where
- Matplotlib: bar, barh, grouped bar, pandas plot

## Key Insights
- Mumbai Indians most successful team with 144 wins
- Winning toss only helps 50.6% — basically a coin flip
- 64% of teams choose to field after winning toss
- Virat Kohli is all time top run scorer
- Yuzvendra Chahal is top wicket taker
- Average IPL match total: 317 runs (158 per team)
