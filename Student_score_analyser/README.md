# 📊 Student Score Analyser

## What This Project Does
Analysis of student exam scores to find performance
patterns, top scorers and subject averages.

## Skills Used
- **Pandas**: iloc, idxmax, idxmin, mean, groupby
- **NumPy**: np.mean with axis parameter
- **Matplotlib**: bar chart for subject averages

## Key Insights
- Identified highest and lowest scoring students
- Calculated pass rate based on average score >= 50
- Found average score per subject across all students

## Files
- analysis.py — full analysis script
- data.csv — student score dataset
