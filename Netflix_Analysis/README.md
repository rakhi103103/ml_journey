# 🎬 Netflix Content Analysis

## What This Project Does
Analysis of 8800+ Netflix titles to find content trends
by country, genre, type, rating and year.

## Dataset
- Source: Kaggle — Netflix Movies and TV Shows
- 8807 titles, 12 features

## Skills Used
- Pandas: groupby, pivot_table, crosstab, explode,
  str.extract, pd.to_datetime, isin, fillna
- NumPy: percentile, boolean masking, np.where
- Matplotlib: bar, barh, pie chart

## Key Insights
- USA dominates with 2818 titles (31% of all content)
- 70% Movies vs 30% TV Shows
- 2019 was peak year for content addition
- Japan & South Korea unique — more TV Shows than Movies
- TV-MA is most common rating — Netflix targets adults
- Average Netflix movie is 99 mins long
